# 채팅 세션 목록 조회

백 진행상황: 완료
설명: 사용자의 채팅 세션 목록을 조회한다. 제목 키워드로 검색할 수 있다.
API 그룹: Chat
URL: /chat/sessions
METHOD: GET
인증 필요: Yes
기능명세서: 대화 히스토리 (https://www.notion.so/31af1fecd5bc80c78fcdd296548fa1b9?pvs=21), 채팅 기록 (https://www.notion.so/327f1fecd5bc80a59691caa35523b7d4?pvs=21), 채팅 세션 이름 검색 (https://www.notion.so/32df1fecd5bc80f5a25dd29084253eca?pvs=21)
담당자: 장민주
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 12일

### 📥 요청(Request)

**Query  Parameters**

| 키 | 타입 | **필수** | 설명 |
| --- | --- | --- | --- |
| keyword | string | X | 채팅방 제목 검색어 (부분 일치) |
| page | number | X | 페이지 번호 (기본: 0) |
| pageSize | number | X | 페이지 크기 (기본: 20) |

### 📤 응답(Response)

**성공 (200 OK)**

```json
{
  "data": [
    {
      "sessionId": number,
      "title": string,
      "sourceType": "normal" | "faq",
      "createdAt": string
    }
  ],
  "totalCount": number,
  "page": number,
  "pageSize": number,
  "totalPages": number
}
```

**실패 (401)** — UNAUTHORIZED

**실패 (500)** — INTERNAL_SERVER_ERROR

**실패 (403)** — SESSION_FORBIDDEN

```json
{
  "error": "SESSION_FORBIDDEN",
  "message": "다른 사용자의 채팅 세션은 접근할 수 없습니다."
}
```
