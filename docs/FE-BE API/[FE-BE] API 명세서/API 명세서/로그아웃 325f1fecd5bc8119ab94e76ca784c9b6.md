# 로그아웃

백 진행상황: 완료
설명: 현재 로그인 세션을 종료하고 토큰을 무효화한다.
API 그룹: Auth
URL: /auth/logout
METHOD: POST
인증 필요: Yes
기능명세서: 로그아웃 (https://www.notion.so/319f1fecd5bc80afbabbc6d6d91998bc?pvs=21)
담당자: 장민주
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 3일

> 인증 필요: `Authorization: Bearer <accessToken>`
> 

### 📥 요청(Request)

> 별도 Request Body 없음
> 

### 📤 응답(Response)

**성공 (200 OK)**

```json
{
  "message": "로그아웃되었습니다."
}
```

> `Set-Cookie: refreshToken=; HttpOnly; Secure; SameSite=Strict; Path=/auth; Max-Age=0`
> 

**실패 (401)** — UNAUTHORIZED

**실패 (500)** — INTERNAL_SERVER_ERROR

### 📜 상세 설명

- 서버에서 PostgreSQL auth_sessions의 해당 유저 세션을 무효화(revoked_at 설정)하여 재발급을 불가능하게 한다
- 응답의 Set-Cookie로 클라이언트의 **Refresh Token Cookie를 제거**한다 (Max-Age=0)
- 클라이언트는 메모리에 저장된 **Access Token도 삭제**하고 로그인 페이지로 이동한다
