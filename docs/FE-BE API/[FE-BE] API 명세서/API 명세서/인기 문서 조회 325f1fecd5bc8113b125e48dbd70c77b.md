# 인기 문서 조회

백 진행상황: 진행 중
설명: 최근 7일 기준 인기 문서 목록을 조회한다.
인기 문서 점수는 클릭 수, 조회 수, 채팅 답변에서의 참조 수를 합산하여 계산한다.
결정 논의사항: 문서 상세 진입 = 조회 수 증가?
출처 링크 클릭 = 클릭 수 증가?
답변에 문서가 source로 포함되면 참조 수 증가?
중복 집계 기준(같은 세션/같은 사용자/짧은 시간 내 중복 제거 여부)
API 그룹: Documents
URL: /documents/popular
METHOD: GET
인증 필요: Yes
기능명세서: 인기 자료 표시 (https://www.notion.so/31af1fecd5bc80df83dfe8d4189d5d70?pvs=21)
담당자: 상진 이
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 13일
