# 채팅 세션 상세 조회

백 진행상황: 완료
설명: 특정 채팅 세션의 메시지 목록을 조회한다.
API 그룹: Chat
URL: /chat/sessions/{sessionId}
METHOD: GET
인증 필요: Yes
기능명세서: 대화 히스토리 (https://www.notion.so/31af1fecd5bc80c78fcdd296548fa1b9?pvs=21)
담당자: 장민주
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 12일

### 📥 요청(Request)

**Path Variable**

| 키 | 설명 |
| --- | --- |
| sessionId | 채팅 세션 ID |

### 📤 응답(Response)

**성공 (200 OK)**

```json
{
  "sessionId": number,
  "title": string,
  "sourceType": "normal" | "faq",
  "messages": [
    {
      "messageId": number,
      "role": "user" | "assistant",
      "content": string,
      "sources": [
        {
          "documentId": number,
          "documentTitle": string,
          "page": number | null
        }
      ],
      "confidence": "high" | "medium" | "low" | null,
      "createdAt": string
    }
  ]
}
```

**실패 (401)** — UNAUTHORIZED

**실패 (404)** — SESSION_NOT_FOUND

**실패 (500)** — INTERNAL_SERVER_ERROR

**실패 (403)** — SESSION_FORBIDDEN

```json
{
  "error": "SESSION_FORBIDDEN",
  "message": "다른 사용자의 채팅 세션은 접근할 수 없습니다."
}
```
