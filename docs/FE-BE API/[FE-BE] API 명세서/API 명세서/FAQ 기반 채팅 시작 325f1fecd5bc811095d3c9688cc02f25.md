# FAQ 기반 채팅 시작

백 진행상황: 시작 전
설명: FAQ를 기반으로 새로운 채팅 세션을 생성한다.
FAQ의 질문과 답변이 초기 메시지로 자동 생성될 수 있다.
결정 논의사항: 후순위
API 그룹: FAQ
URL: /faqs/{faqId}/chat
METHOD: POST
인증 필요: Yes
기능명세서: FAQ에서 바로 채팅으로 넘어가기 (https://www.notion.so/FAQ-31af1fecd5bc80c5bd8cc398b7dfafea?pvs=21)
담당자: 상진 이
프론트 연결 상태: 시작 전

### 📥 요청(Request)

**Path Variable**

| 키 | 설명 |
| --- | --- |
| faqId | FAQ ID |

### 📤 응답(Response)

**성공 (201 Created)**

```json
{
  "sessionId": string,
  "messages": [
    {
      "messageId": string,
      "role": "user",
      "content": string,
      "createdAt": string
    },
    {
      "messageId": string,
      "role": "assistant",
      "content": string,
      "sources": [],
      "confidence": null,
      "createdAt": string
    }
  ]
}
```

**실패 (401)** — UNAUTHORIZED

**실패 (404)** — FAQ_NOT_FOUND

**실패 (500)** — INTERNAL_SERVER_ERROR

### 📜 상세 설명

- 내부적으로 ChatSession 생성 시 source_type = 'faq'로 설정
