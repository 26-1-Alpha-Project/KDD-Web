# 문서 재처리

백 진행상황: 완료
설명: 문서 파싱 실패 시 문서 수정 후 재처리
임베딩 재생성
API 그룹: Admin
URL: /admin/documents/{documentId}/reprocess
METHOD: POST
인증 필요: Yes
기능명세서: 문서 관리 (https://www.notion.so/31af1fecd5bc80f4af30ed8986c9b866?pvs=21)
