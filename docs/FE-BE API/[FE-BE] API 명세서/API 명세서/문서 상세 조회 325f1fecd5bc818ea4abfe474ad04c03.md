# 문서 상세 조회

백 진행상황: 진행 중
설명: 특정 문서의 상세 정보를 조회한다.
채팅 답변 출처 링크 클릭 시 연결될 수 있음
API 그룹: Documents
URL: /documents/{documentId}
METHOD: GET
인증 필요: Yes
기능명세서: 문서 상세 조회 (https://www.notion.so/32cf1fecd5bc8043a6a1f7beb1a40781?pvs=21), FAQ 상세 조회 (https://www.notion.so/FAQ-32df1fecd5bc8046a730da901dcd4234?pvs=21)
담당자: 상진 이
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 13일

### 📥 요청(Request)

**Path Variable**

| 키 | 설명 |
| --- | --- |
| documentId | 문서 ID |
