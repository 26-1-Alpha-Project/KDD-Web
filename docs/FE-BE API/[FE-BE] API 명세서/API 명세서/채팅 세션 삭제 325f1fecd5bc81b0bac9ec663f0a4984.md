# 채팅 세션 삭제

백 진행상황: 완료
설명: 채팅 세션을 즉시 삭제한다.
API 그룹: Chat
URL: /chat/sessions/{sessionId}
METHOD: DELETE
인증 필요: Yes
기능명세서: 채팅 기록 삭제 (https://www.notion.so/31af1fecd5bc80168138e87569850f45?pvs=21)
담당자: 장민주
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 12일

### 📥 요청(Request)

**Path Variable**

| 키 | 설명 |
| --- | --- |
| sessionId | 삭제할 채팅 세션 ID |

### 📤 응답(Response)

**성공 (200 OK)**

```json
{
  "message": "채팅 세션이 삭제되었습니다."
}
```

**실패 (401)** — UNAUTHORIZED

**실패 (404)** — SESSION_NOT_FOUND

**실패 (500)** — INTERNAL_SERVER_ERROR

**실패 (403)** — SESSION_FORBIDDEN

```json
{
  "error": "SESSION_FORBIDDEN",
  "message": "다른 사용자의 채팅 세션은 접근할 수 없습니다."
}
```
