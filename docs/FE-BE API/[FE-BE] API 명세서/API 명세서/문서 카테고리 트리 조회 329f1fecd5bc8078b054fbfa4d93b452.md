# 문서 카테고리 트리 조회

백 진행상황: 완료
설명: [트리 탐색 탭에서 사용]
사전에 정의된 문서 카테고리 트리 구조를 조회한다.
문서 카테고리는 children 형태의 트리 구조로 반환한다.
API 그룹: Documents
URL: /documents/categories
METHOD: GET
인증 필요: Yes
기능명세서: 문서 목록 조회 (https://www.notion.so/319f1fecd5bc811595f0fa29d7cd190c?pvs=21)
담당자: 상진 이
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 9일
