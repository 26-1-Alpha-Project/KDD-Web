# 문서 업로드

백 진행상황: 완료
설명: 관리자가 문서를 업로드한다.
API 그룹: Admin
URL: /admin/documents
METHOD: POST
인증 필요: Yes
기능명세서: 문서 업로드 (https://www.notion.so/319f1fecd5bc81ac94eac1eff2a4c26c?pvs=21)
담당자: 상진 이
프론트 연결 상태: 시작 전
API 명세서 업데이트: 2026년 4월 12일

### 📥 요청(Request)

**formData (file / File)**

| 키 | 설명 |
| --- | --- |
| file | 업로드할 문서 파일 (PDF) |

**formData (data / JSON)**

```json
{
  "title": string,
  "categoryId": number,
  "source": "SW" | "KMU"
}
```

### 📤 응답(Response)

**성공 (201 Created)**

```json

{
  "id": number,
  "title": string,
  "categoryId": number,
  "categoryName": string,
  "status": string,
  "source": string,
  "originalFilename": string,
  "fileSize": number,
  "createdAt": string
}
```

**실패 (400)** — INVALID_INPUT (필수 필드 누락 / 빈 파일)

```json
{
  "error": "INVALID_INPUT",
  "message": "입력값이 올바르지 않습니다."
}
```

**실패 (400)** — INVALID_FILE_TYPE (PDF가 아님)

```json
{
  "error": "INVALID_FILE_TYPE",
  "message": "지원하지 않는 파일 형식입니다."
}
```

**실패 (401)** — UNAUTHORIZED

```json
{
  "error": "UNAUTHORIZED",
  "message": "인증이 필요합니다."
}
```

**실패 (403)** — FORBIDDEN

```json
{
  "error": "FORBIDDEN",
  "message": "관리자 권한이 필요합니다."
}
```

**실패 (404)** — CATEGORY_NOT_FOUND (존재하지 않는 카테고리)

```json
{
  "error": "CATEGORY_NOT_FOUND",
  "message": "존재하지 않는 카테고리입니다."
}
```

**실패 (500)**— INTERNAL_SERVER_ERROR

```
{
  "error": "INTERNAL_SERVER_ERROR",
  "message": "서버 내부 오류가 발생했습니다."
}
```

### 📜 상세 설명

**문서 파이프라인 (BE 책임):**

```
1. BE: 파일 수신 → 서버 로컬 저장
2. BE: PDF 변환 
3. BE: PDF 청킹 (텍스트 추출 + 분할)
4. BE: AI 서버에 POST /api/documents/embed (doc_id = Document.id, 청크 배열 전달)
5. AI: 임베딩 → VectorDB 적재
```
